# 配置文件

## yaml

### 数据类型

#### 基本 key-value

```yaml
k: v
```

#### 数组

```yaml
k1: [v1,v2,v3]
k2: 
  - v1
  - v2
  - v3
```

#### 对象

```yaml
k:
  k1: v1
  k2: v2
  k3: v3
```

```yaml
k: {k1:v1,k2:v2,k3:v3}
```

### 单双引号区别

```yaml
p: '张三\n李四'
```

浏览器得到

```json
{
    "p": "张三\\n李四"
}
```

```yaml
p: "张三\n李四"
```

浏览器得到

```json
{
    "p": "张三\n李四"  // 直接作为字符串会造成换行
}
```
